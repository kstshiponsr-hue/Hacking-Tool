# Shadow Framework v5.0 (Ultimate Edition) 🛡️

**Shadow** হলো টার্মাক্স (Termux) এবং লিনাক্স (Linux) এর জন্য তৈরি একটি অল-ইন-ওয়ান এথিক্যাল হ্যাকিং ফ্রেমওয়ার্ক। এতে ইনফরমেশন গ্যাদারিং থেকে শুরু করে ফিশিং, ওয়েব অ্যাটাক এবং সোশ্যাল মিডিয়া ট্র্যাকিংয়ের মতো প্রায় সব গুরুত্বপূর্ণ টুলস এক জায়গায় সাজানো রয়েছে।

---

## ✨ ফিচারসমূহ (Features)
এই টুলে বর্তমানে ৭টি প্রধান ক্যাটাগরিতে কাজ করা যায়:

1.  **Information Gathering:** Nmap, Whois, IP-Tracer, DNS Lookup.
2.  **Web Attack:** Nikto, SQLMap, Nuclei, Dirb.
3.  **Phishing Attacks:** Zphisher, PyPhisher, Seeker (Location).
4.  **Exploitation:** MetaSploit Setup, Reverse Shell Generator.
5.  **Social Hunter:** Sherlock (Username tracking).
6.  **Password & DDoS:** Hydra (Brute force), Cupp, Hammer DDoS.
7.  **Auto Logging:** প্রতিটি কমান্ডের হিস্ট্রি লগ ফাইলে সেভ থাকে।

## 🚀 ইনস্টলেশন (Installation)

টার্মাক্স ওপেন করে নিচের কমান্ডগুলো একে একে টাইপ করুন:

```bash
# রিপোজিটরি ক্লোন করুন
git clone [https://github.com/kstshiponsr-hue/Hacking-Tool.git](https://github.com/kstshiponsr-hue/Hacking-Tool.git)

# ডিরেক্টরিতে প্রবেশ করুন
cd Hacking-Tool

# ইনস্টলার ফাইল রান করুন (সব প্যাকেজ সেটআপ হবে)
bash installer.sh

# টুলটি চালু করুন
python shadow.py
